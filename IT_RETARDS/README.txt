Параметры запуска.
Используйте любуй интерпритатор, поддерживающий
компиляцию кода на Python 3.

Установка допполнительыных пакетов (не обязательно)
pip install pillow

Реализованная функциональность:
> Возможность ввода логина и пароля
> Работоспособность кнопок, возможность настроить
  их под нужный

Особенность проекта в следующем:
> Хороший и простой дизайн, легко адаптирующийся под будущие
  апдейты.
> Список всех авторизированных пользователей 
  возможность их фильтровать и пробивать в поисковой строке.
> Возможность просмотра текущих действий на устройствах,
  подключенных к режиму киоска (только администратор).
> Видимость всеми пользователями (без прав администратора) их
  доступности к определенным функциям.
> Удобность настроек для администратора.

Основной стек технологий:
> Python 3/tkinter/PIL
> Git
